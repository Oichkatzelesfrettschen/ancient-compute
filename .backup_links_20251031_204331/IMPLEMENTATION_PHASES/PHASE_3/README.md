## KEY INSIGHTS

### 1. Gear Hobbing is THE Constraint
Production of 5,000 digit wheels requires 1,515 continuous machine hours. Even with the fastest 1930s gear hobber (3.3 wheels/hour), this creates a bottleneck:
- Single-shift operation: 26.9 weeks (fails 34-week target)
- 24/7 operation: 13 weeks (meets 34-week target)
- **Decision**: 24/7 operation with 3-shift staffing (only solution)

### 2. Labor Dominates the Budget
- Materials: £7,395 (2.5%)
- **Labor: £177,600 (60.6%)**
- Overhead: £81,600 (27.8%)
- Tools: £820 (0.3%)
- Contingency: £25,869 (8.8%)

This is WHY manufacturing in India is cheaper (lower labor costs) vs. wealthy nations.

### 3. Quality is Achievable with SPC
Statistical Process Control (X-bar & R charts) enables 94-96% first-pass yield using only 1930s machinery. No automation needed.

### 4. Historical Suppliers Actually Existed
- Tata Steel: Founded Aug 26, 1907 ✓
- SKF Bearings: Founded 1907 ✓
- David Brown Ltd.: Founded 1860s ✓
- Brown & Sharpe gear hobbers: Available 1920s ✓

This isn't fantasy. These companies existed and could have supplied components.

### 5. It's Actually Doable
The 34-week timeline with proper bottleneck mitigation is realistic. The £293,284 budget is achievable. The 94-96% yield is attainable. This wasn't impossible—it was just never built.
