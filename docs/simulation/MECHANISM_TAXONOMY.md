# Mechanism Taxonomy

Purpose: normalize categories for emulator design, sourcing, and whitepaper coverage.

Axes
- Representation: discrete (symbolic) vs continuous (analog)
- Control: manual, mechanical-programmed, or general-purpose programmable
- Primary state: tokens/marks, positional digits, gear angles, knot topology, card decks

Categories
1) Record-keeping substrates (discrete, manual)
- Tally marks / tally sticks
- Clay tokens / bullae

2) Manual calculation devices (discrete, manual)
- Counting rods / abacus

3) Analog astronomical / navigation instruments (continuous, manual)
- Antikythera mechanism
- Astrolabe
- Slide rule

4) Mechanical arithmetic machines (discrete, mechanical)
- Pascaline
- Leibniz stepped reckoner

5) Programmable pattern/control systems (discrete, mechanical-programmed)
- Jacquard loom (card control)

6) General-purpose programmable engines (discrete, programmable)
- Babbage Analytical Engine (base)
- Ada Lovelace Bernoulli deck (Note G)
- Difference Engine No. 2 (DE2)

7) Anachronistic / unconventional extensions (discrete/continuous, programmable)
- Cellular automata
- DNA computing
- Reaction-diffusion computing
- Slime mold computing (Physarum)
- Reservoir computing
- Membrane computing (P systems)
- In-materio computing
