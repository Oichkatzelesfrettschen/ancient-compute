# Source Acquisition Priority

Purpose: rank missing primary sources by dependency, rarity, and emulator impact.

High priority (blocks emulator fidelity)
- Antikythera mechanism: AMRP reports and a stable museum object page.
- Quipu datasets: Khipu Field Guide datafiles (IA) + Harvard Dataverse artifacts.
- Pascaline: Pascal 1645 description or museum scans (CNAM or equivalent).
- Leibniz stepped reckoner: manuscript scans or museum object photos; Gallica 1703 binary paper.
- Abacus / rod calculus manuals with worked examples.

Medium priority (improves citations and narrative depth)
- Nine Chapters critical scans and Liu Hui commentary.
- Qin Jiushao (1247) scans for CRT provenance.
- Ishango primary article scans (Heinzelin 1962).
- Astrolabe museum object catalogs (NMAH, British Museum).
- Slide rule museum object catalogs (NMAH, Science Museum).

Low priority (contextual, optional)
- Secondary monographs (Needham, Shen/Crossley/Lun, Strickland/Lewis).
- General computing history lecture notes.
- Additional modern reconstructions beyond Plan 28.
