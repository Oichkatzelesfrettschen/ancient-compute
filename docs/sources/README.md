## How to Access Archived Materials

### Science Museum London
- **Online Search:** https://collection.sciencemuseumgroup.org.uk/
- **Access:** Digital surrogates available; originals by appointment
- **Key Collections:**
  - The Babbage Papers (AA110000003)
  - Drawings of the Analytical Engine (AA110000009)
  - Plan 28 drawings and specifications

### Bodleian Library, Oxford
- **Web:** https://archives.bodleian.ox.ac.uk/
- **Access:** Free readers' tickets for UK residents; international appointments available
- **Key:** Papers of William and Ada Lovelace (1826-1907)

### New York Public Library
- **Contact:** https://archives.nypl.org/cps/22061
- **Access:** Appointment system for rare materials research
- **Collection:** 17 items in Pforzheimer Collection

### For IEEE Papers (Require Institutional Access)
- IEEE Xplore: https://ieeexplore.ieee.org/
- University library subscriptions
- Author requests (often authors grant personal use)

## Key Takeaways

### The Analytical Engine
- Designed by Charles Babbage in the 1830s-1840s
- First general-purpose computing machine ever designed
- Never completed during Babbage's lifetime
- Not built until 1985-2002 by the Science Museum, London
- 8,000+ parts, weighs 5 tons, operates entirely mechanically

### Ada Lovelace's Contribution
- Translated Luigi Menabrea's 1842 French description into English
- Added seven notes that were 3x longer than the original
- Note G contains the first complete computer program
- Calculated Bernoulli numbers using the engine
- Understood the machine's potential beyond pure calculation
- First to explain programming concepts (loops, variables, etc.)

### Modern Status
- Complete technical documentation published by Science Museum (open access)
- Plan 28 project actively building an Analytical Engine (2024 progress ongoing)
- Multiple recreations and educational models exist
- Philosophical and historical scholarship is extensive and growing
