# Experiments

Standalone experiment and benchmark scripts relocated from the former
top-level `experiments/` directory. These are not part of the test suite;
run them manually from the project root.

- **carriage_benchmark.py** -- carriage movement micro-benchmark
- **logistic_map_babbage.py** -- logistic map computed on the Babbage emulator
- **micro_mult_test.py** -- micro-op multiplication exerciser
