## System Architecture

The Babbage Analytical Engine consists of 5 main subsystems:

### 1. Mill (Arithmetic Unit)
Performs arithmetic operations:
- Addition, subtraction, multiplication
- Components: gears, levers, shafts
- Mechanical computation engine
- Precision requirement: Â±0.15mm

### 2. Store (Memory)
Stores variables and intermediate results:
- Column-based storage (up to 1,000 digits)
- Mechanical registers
- Read/write mechanisms
- Precision requirement: Â±0.20mm

### 3. Barrel (Instruction Store)
Stores the program:
- Pegged cylinder (Jacquard-style)
- Encodes operations and variable references
- Fixed or interchangeable barrels
- Mechanical encoding

### 4. I/O (Punched Card Interface)
Inputs and outputs:
- Punched card reader (Hollerith-style)
- Punched card punch
- Number printing mechanism
- Mechanical integration

### 5. Control (Sequencer)
Coordinates operation:
- Timing and synchronization
- Program flow control
- Error detection
- Mechanical timing system
